The Bootstrap Grid

The Bootstrap library includes a responsive, mobile first grid system that appropriately scales up to 12 columns. The following interactive primer will provide you with a brief introduction to the Bootstrap grid and demonstrate how it can be easily integrated into your projects.

The Basics

The Bootstrap grid system is used for creating page layouts through a series of rows and columns that wrap content. Mastering the Bootstrap grid is as simple as mastering the following rules.

Key Ingredients

  • Container
  • Rows
  • Colums

Rules of the Road

  • Rows must be placed within a container.
  • Rows are used to create groups of columns.
  • Content must be placed within columns.
  • Columns are the only immediate children of rows.
  • Columns create gutters via padding.
  • Columns are created by specifying a number up to twelve.

Grid Dimensions

The Bootstrap grid follows a mobile-first approach, beggining with the smallest screen size and progressing in size to accomodate larger viewports.

Device Dimensions

Device Width Container Width Content Width
Extra Small Devices less than 768px auto auto
Small Devices 768px and up 750px ~720px
Medium Devices 992px and up 970px ~940px
